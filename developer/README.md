# developer/ — Developer Documentation

Developer-facing documentation for the ankurnema.in repo.

## What Goes Here

- Setup guides (local dev, environment configuration)
- Contribution guidelines
- Deployment runbooks
- Architecture notes (complement to `docs/adr/`)
- Troubleshooting guides
- CI/CD documentation

## What Does NOT Go Here

- Architecture Decision Records → `docs/adr/`
- Phase completion summaries → `docs/phases/`
- Blog content → `src/content/blog/`
- Component documentation → inline or `src/components/`
